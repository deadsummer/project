Исполняемый файл "window.py" - запускает оконное приложение, в котром нужно ввести две категории для анализа непересекающихся словосочетаний.

Подключаемые модули: parser_sasha , text_analisys, analis

parser_sasha
имеет функцию pars, которая принимает аргументами (список ссылок на категории, текстбокс)
создает количество файлов по количеству данных ссылок со всеми текстами из категории в каждом


text_analisys
содержит функции для разбиения текстов на предложения, слова, основы.


analis - имеет функцию analis без аргументов
составляет матрицу вероятностей сочетаний основ для текста, выполняет малоранговую аппроксимацию, вычитание апроксимированной из исходной матрицы, выделение ключевых словосочетаний.
возвращает [лист по каждому файлу[лист по каждому ключевому словосочетанию[лист всех встречающихся форм ключевого словосочетания]]]
